# 什么是电脑  
电脑，很多人一听觉得电脑不就是台式机或者笔记本吗？其实，我们生活中，电脑无处不在。下面就让我来列举一下吧。 

- 智能手机 （其实大家可能早就知道这是一台电脑了）
- 非智能手机 （难道这也是电脑？）
- Ipad （这个大家也知道）
- 智能电视/机顶盒 （好像也是哈）
- 家里的路由器 （What？这也是？）
- 蓝牙耳机 （不是吧，耳机也是电脑）
- 共享单车的锁 （没错，这也是台 mini 电脑）
- 各式各样的智能手环，智能手表（难道一加上智能两个字，就成了电脑？）

Oh my God! 这些都是电脑。那么电脑究竟是什么？今天，就由小编来仔细介绍一下什么是电脑，看完这一篇文章也许你对电脑就会有了更加全面和深入的理解。

百度百科第一段的描述是这样的：
电脑（Computer）是一种利用电子学原理根据一系列指令来对数据进行处理的机器。电脑可以分为两部分：软件系统和硬件系统。第一台电脑是1946年2月15日在美国宾夕法尼亚大学诞生的ENIAC通用电子计算机。
这里面的机器指的就是硬件，指令就是程序或者软件，而对数据进行处理就是电脑的工作。

而百度百科中对电脑的 “理论基础” 的描述是这样的：
约翰·冯·诺依曼提出的计算机体系结构奠定了现代电脑的结构理论基础：

1. 计算机由运算逻辑单元、控制器、输入和输出设备、记忆单元五大单元组成；
2. 程序存储控制；
3. 以二进制为计算机基本单位。

这里面的概念 2 和 3 由于比较虚今天不作讨论。我们就来讨论概念 1。目前的电脑，这五大单元中的运算逻辑单元和控制单元合二为一，就是大名鼎鼎的 CPU 了。

下面，就让我们来看看这些电脑的单元组成吧。

**台式电脑/笔记本电脑**

CPU （运算逻辑单元和控制单元）
当然就是 Intel 或者 AMD 的CPU啦。
输入
键盘和鼠标，麦克风，触摸屏等，以太网卡,WIFI无线网卡
输出
显示器，扬声器，以太网卡,WIFI无线网卡
存储器
内存，硬盘，光盘等

这里的内存和硬盘有根本的不同。内存是一种访问速度非常快，但一旦关机后里面存储的数据就消失了。而硬盘里面的数据访问起来速度比较慢，但关机后里面存储的数据不会消失。


**智能手机**

CPU （运算逻辑单元和控制单元）
发烧的最新一代小米手机总是号称用的骁龙的最新一代 CPU，可见 CPU 多么重要。CPU 的强弱直接决定你打开微信，QQ，支付宝 等应用的速度。而 CPU 中还集成了另外一个计算单元 GPU，专门用于图形图像方面的计算。这部分的强弱直接决定你玩王者荣耀卡不卡。
输入
触摸屏和大部分手机下巴上的按钮，麦克风，3G/4G/WIFI 无线连接
输出
屏幕，扬声器，3G/4G/WIFI 无线连接
存储器
就是内存和存储。比如小米6 6G/128G 里面的 6G 就是内存，而128G 的就是存储，相当于电脑的硬盘。

Ipad，智能电视和智能手机的结构基本一样，只是屏幕大点而已
机顶盒和路由器和智能手机的结构也基本一样，只是没有屏幕
蓝牙耳机里面也是这样，不过他们的CPU比较弱，只需要用来处理蓝牙连接，并且把蓝牙传过来的声音信号从耳机的扬声器里面播放出来。
智能手环/手表同样，只是屏幕比较小而已

看，这么多电脑。虽然样子大不相同，但里面的结构基本相似，是不是很神奇？
